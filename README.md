## Setup

  ##### Clone the repository
```bash
git clone https://github.com/ishandeveloper/Quizzy-App.git
```
  ##### Move to the desired folder
```bash
cd \Quizzy-App
```

  ##### To run the app, simply write
```bash
flutter run
```
